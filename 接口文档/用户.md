# api/user/verify-account

验证是否已注册

传入：

| 键      | 含义 | 类型   |
| -------- | ------- | ------ |
| username | 用户名 | String |
| email | 邮箱 | String |
| phone | 手机号 | String |

返回：

| 键      | 含义 | 类型   |
| -------- | ------- | ------ |
| error | 错误代码     | Number |

---

说明：

键phone：格式为区号+11位电话号码（没有前导0）

键error（错误代码）

| 值    | 含义    |
| ------ | ---------- |
| 0   | 用户未注册       |
| 1   | username，email和phone中有超过1个非空键 |
| 2   | 用户已注册 |

# api/user/set-own-info

设置已登录用户自身的信息，其中设置邮箱和手机号需要验证码

传入：

| 键    | 含义    | 类型   |
| ------ | ---------- | ------ |
| username   | 用户账户       | String |
| name   | 用户昵称       | String |
| phone   | 用户手机号码       | String |
| gender   | 用户性别       | String |
| mail   | 用户邮箱       | String |
| avatar | 用户头像文件     | String |
| birthday | 用户生日     | String |
| code | 验证码     | Number |
| original_code | 原手机验证码     | Number |

返回：

| 键    | 含义    | 类型   |
| ------ | ---------- | ------ |
| error | 错误代码     | Number |

---

说明：

键error（错误代码）

| 值    | 含义    |
| ------ | ---------- |
| 0   | 无错误       |
| 1   | 用户未处于登录状态       | 
| 2   | username、name、phone、gender、mail、avatar、birthday都为空       |
| 3   | phone和email同时不为空       | 
| 4   | phone和email中有一个为空另一个不为空，但是code或original_code为空 | 
| 5   | phone和email中有一个为空另一个不为空，但是code或original_code错误 | 
| 6   | phone和email中有一个为空另一个不为空，但是code或original_code过期 | 
| 7   | username、name、avatar中有某一个键内容违规或格式错误       |
| 8   | username、name、phone或email已注册       |
| 9   | phone修改次数过多       |

# api/user/add-follow

关注用户

传入：

| 键    | 含义    | 类型   |
| ------ | ---------- | ------ |
| id    | 用户id    | String |

返回：

| 键    | 含义    | 类型   |
| ------ | ---------- | ------ |
| group_name    | 关注分组    | Number |
| error | 错误代码     | Number |

---

说明：

键error（错误代码）

| 值    | 含义    |
| ------ | ---------- |
| 0   | 无错误       |
| 1   | 用户已在该组中被关注       | 
| 2   | 用户不存在       | 
| 3   | 用户未处于登录状态       | 
| 4   | 该组不存在       | 

键group（关注分组）：默认加入默认分组，若加入的用户在其他组中存在，则从其他组删除，再将其加入新组

| 值    | 含义    |
| ------ | ---------- |
| special   | 特别关注分组       |
| default   | 默认分组       | 
| 其他值   | 其他分组       |
